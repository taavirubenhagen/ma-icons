**Ma Icons**

*Ultra-minimalist UI icon pack.
Still in development.*

<br/>

**Usage**

```
<script src="https://unpkg.com/ma-icons@0.12.x/dist/ma-icons.iife.js"></script>
```

```
<ma-icon
    name="info-topright"
    size="16"
    weight="5"
    color="hsl(0deg 0% 0%)"
></ma-icon>
```
